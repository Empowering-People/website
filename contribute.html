Empowering you to help improve our site

Noticed a typo somewhere? Or maybe we've missed something in terms of accessibility. Or perhaps it's something completely different. Regardless of what it is, we empower you to directly contribute to our website!

This site's code is fully open source and publicly viewable on our GitHub. If you have anything you would want to open a discussion, issue, or pull request about, no matter how small or trivial it may seem, it's all welcome, and will not go disregarded by us. It doesn't matter if you're an experienced web developer suggesting sweeping changes to our site's entire codebase, or someone who doesn't know the first thing about coding, simply making a suggestion or letting us know about an error we made.

Note that contributing is entirely voluntary, you don't have to if you don't want to, and if you do, you shouldn't expect any more in return than a contributor credit thanking you and linking to your GitHub account (of course, if you don't want to be credited, all you have to do is say so).

The technical details

This site is hosted on GitHub Pages, and uses Jekyll, on top of vanilla HTML, CSS, and JS, so familiarity with these is preferred if you're wanting to make direct changes to our code, though new developers are naturally welcome.

When it comes to JS, we aim to rely on it as little as possible, for a few reasons:

  • If someone viewing this site is blocking JS in their browser, it won't break things too badly.
  • Minimal JS means faster loading times than many other websites.
  • Smaller amounts of JS are more easily audited by those concerned about the code they're letting run in their browsers.
  • Gabriel Sykes, the Empowering People employee who maintains this website, hates working with JS and would rather keep it to small amounts of "set it and forget it" code where it is absolutely necessary to make something work (like the dyslexia-friendly font toggle).


Familiarity with Git is also advised, as this site makes heavy use of it for version control, and is hosted on a service provided by GitHub.
